# Лесная Линия — демонстрационный лендинг

Готовый статический сайт строительной компании в тёмном премиальном стиле.

## Запуск

Откройте `index.html` в браузере или запустите локальный сервер:

```bash
python3 -m http.server 8080
```

После этого откройте `http://localhost:8080`.

## Структура

- `index.html` — разметка всех секций.
- `styles.css` — адаптивный дизайн для desktop/tablet/mobile.
- `script.js` — меню, модальное окно, формы, анимации появления.
- `assets/` — локальные изображения и favicon.

Формы работают в демонстрационном режиме и показывают уведомление об успешной отправке. Для реального проекта подключите CRM, Telegram-бота, почтовый сервис или серверный обработчик.

Фотографии: Pexels, Harrison Haines и Introspective Design.
